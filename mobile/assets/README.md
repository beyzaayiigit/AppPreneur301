# Varlıklar ve lisans

## LUT (.cube) — gelecek sürüm

Gerçek 3B LUT dosyaları bu klasöre eklendiğinde:

1. Ticari veya açık kaynak lisans metnini `assets/licenses/` altında saklayın.
2. Üretim APK/IPA’sında lisansın yeniden dağıtım şartlarına uyduğunuzu doğrulayın.
3. Dosya adlarını `src/engine/presets.ts` ile eşleştirin.

## Grain doku (opsiyonel)

Harici film grain görüntüsü kullanırsanız telif/lisansı burada belgelendirin. MVP’de prosedürel grain kullanılıyorsa harici doku gerekmez.
